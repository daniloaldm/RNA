# RNA

## Instalação e Execução
Clone o repositório:
```
git clone git@github.com:daniloaldm/RNA.git
```
Vá para o diretório:
```
cd RNA
```
Execute:
```
python3 -m venv .
. bin/activate
pip3 install -r requirements.txt
```

Para executar os arquivos:
```
cd atv1
python3 Perceptron_01.py
```
Para executar os arquivos no jupiter:
```
cd atv1
jupyter-lab
```
